# yamldiff
`yamldiff` diffs between 2 yaml files

### Installation

```bash
go install kmodules.xyz/yamldiff@latest
```

### Usage

```bash
yamldiff src.yaml dst.yaml [-t json|strategic] [-o yaml|json]
```
